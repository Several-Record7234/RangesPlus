# RangesPlus

An Owlbear Rodeo extension for a simple rangefinder tool that supports:
- square and circular rings for top-down maps
- rhombus and ovoid rings isometric and dimetric maps

This operates mostly the same as the original extension (found at https://extensions.owlbear.rodeo/ranges) but with automatic detection of the map's grid type (square, isometric, dimetric), and a manual override in the settings 'cog' dropdown menu.
